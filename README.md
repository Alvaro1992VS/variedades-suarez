# 🧺 Variedades Suárez — Tienda Virtual

¡Bienvenido al repositorio oficial de **Variedades Suárez**! Esta es una aplicación web moderna y optimizada, diseñada en Python con **Streamlit** para facilitar el encargo y envío de productos directo hasta la puerta de la casa de los clientes en el pueblo.

---

## 🚀 Características de la Aplicación

* **📱 Diseño Adaptable:** Totalmente optimizado para teléfonos móviles y compatible con **Modo Claro y Modo Oscuro**.
* **🛍️ Catálogo por Categorías:** Navegación fluida mediante pestañas horizontales para encontrar productos rápidamente.
* **🔍 Buscador Inteligente:** Lupa de filtrado en tiempo real por nombre de producto.
* **🎟️ Sistema de Cupones:** Motor de descuentos dinámico (ej: `SUAREZ10`).
* **🧾 Recibo en Pantalla:** Desglose detallado estilo ticket de caja para el cliente antes de comprar.
* **🟢 Integración con WhatsApp:** Envío automatizado de los datos de entrega y el carrito formateado con un solo botón.

---

## 🛠️ Tecnologías Utilizadas

* **Python 3**
* **Streamlit** (Interfaz de usuario)
* **HTML5 / CSS3** (Estilos personalizados para la compatibilidad de color)

---

## 📦 Estructura del Proyecto

El repositorio está organizado de la siguiente manera para su correcto despliegue en **Streamlit Community Cloud**:

* `streamlit_app.py`: El código fuente principal de la aplicación.
* `requirements.txt`: Las dependencias de librerías necesarias para el servidor.
* `*.jpg`: Archivos de imágenes y fotos de los productos en catálogo.

---

## 💰 Métodos de Pago y Despliegue
* **Pago:** Únicamente pago en efectivo al recibir la mercancía.
* **Estado del Servidor:** Activo y sincronizado automáticamente con los cambios de este repositorio.
